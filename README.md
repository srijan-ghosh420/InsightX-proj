# InsightX Analytics: Text-to-SQL Engine

An AI-powered interface that converts natural language questions into executable SQL queries using Gemini. 

## 🛠️ Tech Stack
- **AI Model:** Gemini 2.5 Flash
- **Database:** SQLite
- **Languages:** Python
- **Libraries:** `google-genai`, `pandas`, `python-dotenv`
